# Learn Java with Projects – Practice Repository
This repository contains my Java practice programs, exercises, and mini-projects while learning from the book:

"Learn Java with Projects" by Maaike Van Putten and Dr. Seán Kennedy.

The goal of this repository is to:
- Strengthen Java fundamentals
- Practice object-oriented programming
- Build problem-solving skills
- Maintain consistency through daily coding
- Track my progress publicly
